Gerador de Scripts para PDFtk Server

Pré-requisito: Você precisa ter o PDFtk Server instalado e configurado no PATH do seu sistema. Salve e execute o script gerado na mesma pasta dos seus arquivos PDF.

2. Extrair um Intervalo de Páginas
3. Dividir a Cada 'X' Páginas
4. Dividir em Páginas Únicas (Burst)
5. Rotacionar Páginas
